# DVA.FULL Flows — Multi-Agent Tier

DVA.FULL coordinates multiple agents for complex, long-running tasks. It includes planning, execution, and recovery layers while staying under Sentinel governance.

## Flows

- `dva_full_multiagent.json` — Entry point for multi-agent task orchestration
- `dva_full_recovery_protocol.json` — Ensures failed tasks are retried or escalated
- `.md` files — Document flow responsibilities and safety expectations

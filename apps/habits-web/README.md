# Mobius Habits Web

**C-150: Daily Reflections + Citizen Shield Frontend**

Next.js frontend for Mobius Habits testnet app.

## Pages

- `/` - Homepage
- `/reflections` - Daily reflection form
- `/shield` - Weekly Citizen Shield checklist

## Environment Variables

- `NEXT_PUBLIC_HABITS_API_BASE` - Habits API base URL (default: http://localhost:4005)

## Development

```bash
npm install
npm run dev
```

App runs on http://localhost:3006

## Build

```bash
npm run build
npm start
```

## Features

- Daily reflection submission (3 questions)
- Weekly Shield checklist (5 steps)
- MII score display
- MIC points tracking
- Streak visualization

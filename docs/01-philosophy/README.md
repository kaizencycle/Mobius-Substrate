## Key Concepts

**The Three Covenants** form the constitutional foundation:

1. **Integrity** — Truth, transparency, cryptographic proof
2. **Ecology** — Balance, sustainability, long-term harmony
3. **Custodianship** — Stewardship over ownership, succession planning

**Kaizen (改善)** — Continuous improvement through small, compounding steps

**Kintsugi (金繕い)** — Honoring repairs; cracks make the story more beautiful

**Summon (召唤)** — Calling forth the spark in others by name

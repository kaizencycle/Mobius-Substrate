# Technical Steering Committee (TSC)

Owns kernel, specs, reference libs, releases, security response. Members appointed by merit + election. Maintains ROADMAP and accepts RFCs.

## Responsibilities

- Technical roadmap and architecture decisions
- Release management and versioning
- RFC review and acceptance
- Security response and vulnerability management
- Reference implementation maintenance
- Test and CI/CD standards

## Current TSC Members

*To be elected upon Foundation formation*

## Decision Process

- Proposals via RFCs
- Discussion period (≥14 days)
- Vote (simple majority for routine, ⅔ for breaking changes)
- Implementation and documentation

\section{Classification System}

The Mobius Systems documentation follows a hierarchical classification:

\begin{description}
    \item[000] Meta-Documentation — Indices, navigation guides, documentation system
    \item[100] Introduction — Getting started, handbooks, onboarding
    \item[200] Theoretical Foundations — Philosophy, theory, manifestos, labor theory
    \item[300] Governance \& Policy — Constitution, governance structures, policies
    \item[400] Technical Architecture — System design, DVA, MEMT, integrity ledger
    \item[500] Implementation — Deployment guides, development guides, operations
    \item[600] Operations — Procedures, runbooks, rituals, protocols, drift control
    \item[700] Research \& Publications — Whitepapers, research, blueprints, encyclopedia
    \item[800] Reference — APIs, schemas, quick references, intelligence classification
    \item[900] Appendices \& Archives — Supporting materials, historical documents
\end{description}

- Accent: #00d4aa (teal/cyan) — growth, balance

- Code: Monospace, syntax-highlighted

**Imagery:**
- Geometric patterns (Mobius strips, cycles)

**Core Characteristics:**

- **Visionary but Grounded** — 50-year vision with today's milestones
- **Serious but Human** — Important work, human connection

**What We Avoid:**
- Hype and overpromising
- Corporate marketing speak
- Techno-utopianism without nuance
- Jargon without explanation

## Crisis Communications

If a security incident or controversy arises:

1. **Acknowledge Immediately** — Don't go silent
2. **Be Transparent** — Share what we know, when
3. **Take Responsibility** — Own mistakes, no blame-shifting
4. **Communicate Plan** — What we're doing to fix it
5. **Follow Through** — Update as situation evolves
6. **Learn Publicly** — Post-mortem and improvements

See [`../05-security/incident-response.md`](../05-security/incident-response.md) for procedures.

# Quorum & Voting Rules

Council quorum ≥5; TSC quorum ≥50%+1. Supermajority (⅔) for kernel or license/ethical changes. Emergency votes permitted for critical security.

## Vote Types

### Simple Majority (>50%)
- Routine technical decisions
- Working group formation
- Non-breaking releases

### Supermajority (≥⅔)
- Kernel breaking changes
- License or ethical addendum changes
- Council/TSC member removal
- Charter amendments

### Unanimous
- Emergency security overrides of standard process
- Changes to voting rules themselves

## Emergency Procedures

In critical security situations:
- TSC may act with 48-hour notice
- Must document rationale
- Retrospective Council review within 7 days

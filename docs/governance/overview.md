- **Concord Council**: Ethics, policy, anti-capture oversight
- **Technical Steering Committee (TSC)**: Technical roadmap, releases, kernel
- **Working Groups**: Security, Attestation, Federation, Education
- **Community**: Contributors, adopters, researchers

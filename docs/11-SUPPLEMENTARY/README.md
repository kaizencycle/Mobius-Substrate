# Supplementary Documentation

This directory contains supplementary documentation organized by category.

## Structure

- **echo-layer/** - ECHO Layer implementation documentation
- **implementation-summaries/** - Cycle implementation summaries (C-150, etc.)
- **root-docs/** - Documentation moved from repository root
- **architecture-docs/** - Architecture, formal verification, migration guides, etc.

## Organization

Documentation has been organized from the repository root to keep the main directory clean while preserving all important information.

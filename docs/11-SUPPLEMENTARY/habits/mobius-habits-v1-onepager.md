# Mobius Habits v1 — Civic Pilot One-Pager

**Daily Reflections. Weekly Cyber Hygiene. A Soft Launch for Proof-of-Integrity.**

## 1. Purpose

Mobius Habits v1 is the first public surface of the Mobius Systems ecosystem — the "small window" into a powerful civic operating system built on Proof-of-Integrity.

It provides:
- **Daily Reflections** (Strange Metamorphosis Loop)
- **Weekly Cybersecurity Hygiene** (Citizen Shield)

These two lightweight rituals generate integrity signals that feed into the Mobius Integrity Index (MII) and future MIC (Mobius Integrity Credit) tokenomics.

This soft launch is intentionally minimal, human-focused, and safe for real-world pilots.

## 2. Why This Matters

This app serves three functions simultaneously:

### A. Civic Habit Formation

Regular reflections and cyber hygiene reduce:
- Anxiety
- Avoidance
- Digital vulnerability
- Fragmented worldviews

### B. Safe, Controlled Data for MII / MIC Testnet

Mobius collects low-risk, non-behavioral reflections data that trains the Proof-of-Integrity engine without:
- Surveillance
- Social profiling
- Large-scale behavioral analytics

### C. A Practical On-Ramp to Digital Citizenship

Users start forming:
- A stable Mobius ID
- A personal integrity profile
- Streaks and rituals that map into future AGI governance

This becomes the seed population for Mobius Testnet 1k.

## 3. Core Features in v1

### Daily Reflections (SML v1)

Users answer:
1. **Worldview** — "What did you notice about the world today?"
2. **Mood Check-In** — One-word + slider (0–1)
3. **Tomorrow's Intent** — A micro-commitment

Backend computes:
- Echo Score
- GI Score
- Intent Category
- 7-day streak

### Citizen Shield (Weekly Security Ritual)

Users complete 5 steps:
- OS/app updates
- Router hygiene
- Browser lockdown
- 2FA
- Backup + recovery

Generates a Shield Integrity Score (0–1).

## 4. Data & Privacy Philosophy

Mobius Habits collects:
- No biometrics
- No location
- No behavioral surveillance
- No social graph
- No browsing data

All signals come from the user's own written reflection or intentional actions, not extracted data.

This is human sovereignty by design.

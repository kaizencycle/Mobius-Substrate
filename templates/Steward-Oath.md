## Usage

### When to Recite the Full Oath

1. **Upon becoming a Steward**
   - Appointment or election ceremony
   - First Command Ledger entry

2. **At the start of each Epoch**
   - Epoch transition ceremony
   - Public recommitment

3. **During crisis recovery**
   - Return from Tier R2 or R3
   - After GI_state restoration

4. **At major festivals**
   - City-State founding anniversary
   - Inter-City-State gatherings

### When to Use the Short Oath

1. **Daily check-ins**
   - Personal reminder
   - Dashboard header

2. **Beginning each Command Ledger cycle**
   - Header quote

3. **Public communications**
   - Social media profiles
   - Email signatures
   - Public reports

4. **Steward badges and credentials**
   - Visual identity
   - Cathedral certification

## Oath Violations

### What Breaks the Oath

1. **Lying about metrics**
   - Falsifying GI_state
   - Hiding integrity violations

2. **Self-serving decisions**
   - Personal enrichment over city welfare
   - Nepotism

3. **Corruption**
   - Bribery
   - Manipulation
   - Fraud

4. **Abandonment**
   - Neglecting duties
   - Ignoring citizen needs
   - Refusing to communicate

### Consequences

- Automatic Tier relegation
- Steward probation or removal
- Cathedral investigation
- Public transparency report

### Restoration

- Acknowledgment of violation
- Remediation plan
- GI restoration protocol
- Re-taking of Oath (if permitted)
